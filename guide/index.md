# Обзор системы xkurier

**xkurier** — полноценная система доставки продуктов, состоящая из трёх приложений и серверной части.

## Компоненты системы

| Приложение           | Для кого                 | Платформы    |
| -------------------- | ------------------------ | ------------ |
| **xcourier**         | Клиенты (покупатели)     | iOS, Android |
| **xcourier_manager** | Менеджеры магазинов      | iOS, Android |
| **xcourier_admin**   | Администраторы платформы | iOS, Android |

## Как это работает

1. **Клиент** открывает приложение xcourier, выбирает магазин, добавляет товары в корзину и оформляет заказ.
2. **Менеджер магазина** получает уведомление о новом заказе в xcourier_manager, собирает заказ и меняет его статус.
3. **Администратор** через xcourier_admin управляет магазинами, менеджерами, баннерами и видит аналитику по заказам.

## Серверная часть

- **Firebase Firestore** — основная база данных (real-time синхронизация)
- **Firebase Auth** — аутентификация пользователей
- **Firebase Cloud Functions** — бизнес-логика (создание заказов, расчёт доставки, push-уведомления)
- **Yandex Cloud Object Storage** — хранение изображений товаров

## Статусы заказа

Каждый заказ проходит через цепочку статусов:

| Статус     | Описание                             |
| ---------- | ------------------------------------ |
| Создан     | Заказ только что оформлен клиентом   |
| Собирается | Менеджер начал сборку заказа         |
| Собран     | Заказ собран, ожидает курьера        |
| В пути     | Курьер забрал заказ и едет к клиенту |
| Завершен   | Заказ успешно доставлен              |
| Отменен    | Заказ отменён                        |

## Способы оплаты

Система поддерживает два способа оплаты:

- **Наличные** — оплата курьеру при получении
- **СБП** — оплата с карты через Систему быстрых платежей (в разработке)

## Способы доставки

- **Курьерская доставка** — доставка по указанному адресу
- **Самовывоз** — самостоятельный забор из магазина
