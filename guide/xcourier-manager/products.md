# Управление товарами

## Каталог товаров

Страница товаров показывает весь ассортимент магазина менеджера. Доступны режимы отображения:
- **Сетка** — карточки с изображениями
- **Список** — компактный вид с основной информацией

## Фильтрация

Товары можно фильтровать:
- По категориям (отдельная страница выбора категорий)
- По наличию (страница «Нет в наличии» для быстрого управления стоком)
- Поиск по названию

## Добавление товара

Для создания нового товара менеджер заполняет:
- Название товара
- Цена (в тенге)
- Вес/объём
- Описание (опционально)
- Категории (мультивыбор)
- Фотография товара (загружается через Cloud Function `uploadImage`, конвертируется в WebP)
- Флаг наличия (inStock)

## Редактирование товара

Менеджер может изменить любое поле существующего товара:
- Обновить цену
- Сменить изображение (старое автоматически удаляется из хранилища)
- Изменить категории
- Переключить наличие

## Управление наличием

Отдельная страница «Нет в наличии» показывает только товары с `inStock: false`. Менеджер может быстро вернуть товар в наличие переключателем.

## Категории

Доступна отдельная страница управления категориями магазина:
- Создание новой категории
- Изменение порядка сортировки
- Активация/деактивация категории
