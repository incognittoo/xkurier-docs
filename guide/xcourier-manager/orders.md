# Управление заказами

## Список заказов

Основной экран приложения — список активных заказов магазина. Заказы обновляются в реальном времени через Firestore listener.

Каждая карточка заказа содержит:
- Номер заказа
- Имя и телефон клиента
- Адрес доставки
- Текущий статус (цветной бейдж)
- Способ оплаты
- Сумма заказа
- Дата создания

## Архив заказов

Через меню доступна страница «Все заказы» с полной историей, включая завершённые и отменённые.

## Детали заказа

На странице деталей менеджер видит:
- Полную информацию о заказе
- Список товаров с изображениями, ценами и количеством
- Контактные данные клиента
- Адрес доставки и комментарий

## Смена статуса

Менеджер может менять статус заказа через dropdown-меню:

| Переход | Действие |
|---|---|
| Создан → Собирается | Начал сборку заказа |
| Собирается → Собран | Заказ собран, ждёт курьера |
| Собран → В пути | Курьер забрал заказ |
| В пути → Завершен | Заказ доставлен клиенту |
| Любой → Отменен | Отмена заказа |

При каждой смене статуса клиент получает push-уведомление.

## Редактирование заказа

Менеджер может отредактировать заказ:
- Изменить количество товара в заказе
- Удалить товар из заказа
- Добавить новый товар из каталога магазина

Редактирование доступно до завершения или отмены заказа.
