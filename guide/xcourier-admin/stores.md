# Магазины

## Список магазинов

Администратор видит все магазины платформы с информацией:
- Название и логотип
- Адрес
- Статус (открыт, закрыт, временно закрыт, скрыт)
- Количество заказов за неделю

## Создание магазина

Для создания нового магазина заполняются:
- Название
- Телефон
- Адрес (с автоподсказками и геокодированием)
- Координаты (широта, долгота) — определяются автоматически по адресу
- Логотип
- Время доставки (мин/макс в минутах)
- Расписание работы по дням недели

## Редактирование магазина

Все поля магазина можно изменить. Дополнительно доступно:
- Переключатель `active` — скрыть/показать магазин в клиентском приложении
- Переключатель `temporarilyClosed` — временно закрыть магазин

## Расписание работы

Для каждого дня недели задаётся:
- Время открытия
- Время закрытия
- Признак выходного дня

Расписание влияет на статус магазина — вне рабочих часов магазин автоматически показывается как «Закрыт».

## Категории магазина

Из карточки магазина можно перейти к управлению его категориями:
- Создание категорий
- Изменение порядка сортировки
- Включение/отключение категорий
- Просмотр количества товаров в категории

## Поиск адреса

При создании или редактировании магазина доступен экран поиска адреса с автоподсказками. Координаты определяются автоматически из выбранного адреса.
