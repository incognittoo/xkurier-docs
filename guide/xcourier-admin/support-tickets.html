Обращения в поддержку ​

Список обращений ​

Администратор видит все обращения от пользователей, отсортированные по дате обновления. Каждое обращение отображается с:

  • Заголовок
  • Статус (цветной бейдж)
  • Дата создания

Статусы обращений ​

СтатусОписание
ОткрытНовое обращение, ожидает рассмотрения
В обработкеАдминистратор занимается вопросом
ЗакрытОбращение обработано

Работа с обращением ​

На странице деталей администратор может:

  • Прочитать полное описание проблемы
  • Изменить статус обращения
  • Написать ответ пользователю

При ответе или смене статуса пользователь получает push-уведомление через Cloud Function onSupportTicketUpdated.
