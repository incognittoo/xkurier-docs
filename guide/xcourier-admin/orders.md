# Заказы и отчёты

## Просмотр заказов

Администратор видит заказы по всем магазинам платформы. Заказы можно фильтровать:
- По статусу (Создан, Собирается, В пути, Завершен, Отменен)
- По магазину

Каждая карточка заказа содержит: номер, статус, имя клиента, магазин, сумму и дату.

## Детали заказа

На странице деталей доступна полная информация:
- Данные клиента (имя, телефон, пол, адрес)
- Магазин и его адрес
- Список товаров с количеством и ценами
- Способ оплаты
- Суммы: товары, доставка, итого

## Аналитические отчёты

Отдельный экран отчётов по заказам с гибкой фильтрацией:

### Периоды
- Сегодня
- 7 дней
- 30 дней
- Этот месяц
- Произвольный период

### Фильтр по магазину
Отчёт можно построить по конкретному магазину или по всем сразу.

### Метрики
- Общее количество заказов
- Разбивка по статусам (создан, собирается, в пути, завершён, отменён)
- Общая сумма заказов
- Средний чек
- Сумма доставок

### Разбивка по магазинам
Для каждого магазина:
- Количество заказов (всего, завершённых, отменённых)
- Сумма заказов

### Разбивка по способам оплаты
Для каждого способа оплаты:
- Количество заказов
- Сумма
