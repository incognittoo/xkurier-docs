# Админ-панель (xcourier_admin)

Мобильное приложение для администраторов платформы xkurier. Полный контроль над магазинами, менеджерами, контентом и аналитикой.

## Для кого

Владельцы бизнеса и администраторы платформы, которым нужен полный контроль над всеми аспектами системы.

## Основные возможности

- Просмотр всех заказов по всем магазинам
- Аналитические отчёты по заказам
- Управление магазинами (создание, редактирование, расписание)
- Управление менеджерами (создание, привязка к магазину, блокировка)
- Управление баннерами на главной странице клиентского приложения
- Управление категориями товаров
- Обработка обращений в поддержку
- Настройки платежей (наличные, СБП)
- Блокировка пользователей

## Авторизация

Администратор входит по email и паролю. Доступна регистрация нового администратора. Если аккаунт заблокирован, доступ ограничен.

## Навигация

Приложение использует боковой Drawer с разделами:
- Заказы
- Отчёты по заказам
- Магазины
- Обращения в поддержку
- Настройки (менеджеры, баннеры, категории, платежи, заблокированные пользователи)

## Технические экраны

Приложение включает технические экраны для edge-case сценариев:
- **Force Update** — принудительное обновление при выходе критической версии
- **Maintenance** — экран техобслуживания
- **No Internet** — экран отсутствия интернета
- **Error** — экран общей ошибки

Эти экраны управляются через Firebase Remote Config.
