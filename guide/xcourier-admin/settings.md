# Настройки

## Хаб настроек

Раздел настроек объединяет несколько подразделов:

### Способы оплаты

Настройка доступных способов оплаты:
- **Наличные** — включить/выключить
- **СБП** — включить/выключить, указать номер телефона и банк для перевода

Настройки применяются глобально ко всем магазинам.

### Менеджеры

Управление менеджерами — подробнее на странице [Менеджеры](./managers).

### Глобальные категории

Управление категориями, которые применяются ко всем магазинам одновременно. Позволяет стандартизировать категории товаров по платформе.

### Баннеры

Управление рекламными баннерами — подробнее на странице [Баннеры](./banners).

### Заблокированные пользователи

Список заблокированных пользователей клиентского приложения. Администратор может:
- Просмотреть список заблокированных
- Разблокировать пользователя
