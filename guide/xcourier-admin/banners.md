# Баннеры

## Что это

Баннеры отображаются в карусели на главной странице клиентского приложения xcourier. Используются для промо-акций, рекламы магазинов и внешних ссылок.

## Список баннеров

Администратор видит все баннеры с превью изображения, статусом активности и порядком сортировки.

## Создание баннера

Поля:
- **Изображение** — загружается через Cloud Function `uploadImage`, конвертируется в WebP
- **Тип таргета** — определяет что происходит при нажатии:
  - `none` — ничего (просто картинка)
  - `web` — открывает внешнюю ссылку
  - `store` — переходит к магазину в приложении
- **URL/ID таргета** — ссылка или ID магазина (в зависимости от типа)
- **Порядок** — числовое значение для сортировки
- **Активность** — включён/выключен

## Редактирование баннера

Все поля доступны для редактирования. При смене изображения старое автоматически удаляется из Yandex Object Storage.
