# Каталог и товары

## Просмотр товаров

Товары отображаются на странице магазина в виде сетки карточек. Каждая карточка содержит:

- Изображение товара
- Название
- Вес/объём
- Цена (в тенге)
- Кнопка добавления в корзину

Товары можно фильтровать по категориям магазина. Категории отображаются горизонтальным списком чипов над товарами.

## Детальная страница товара

При нажатии на карточку открывается полноэкранная страница товара с анимацией перехода (slide up + fade). На странице:

- Крупное изображение товара
- Название, вес, описание
- Цена
- Кнопка добавления в корзину с выбором количества

## Поиск товаров

Поиск доступен по иконке на главной странице и на странице магазина. Поиск работает по названию товара с помощью поисковых токенов, сгенерированных Cloud Function.

Во время набора текста результаты обновляются в реальном времени. Показывается скелетон-анимация пока идёт загрузка.

## Доступность товаров

Каждый товар имеет два флага:

- **isAvailable** — товар доступен для показа в каталоге
- **inStock** — товар есть в наличии на складе

Товары, которых нет в наличии, могут отображаться в каталоге, но не могут быть добавлены в корзину.

## Пагинация

Список товаров поддерживает пагинацию — при прокрутке вниз подгружается следующая порция данных. Это обеспечивает быструю загрузку даже для магазинов с большим ассортиментом.
