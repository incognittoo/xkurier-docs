# Избранное

## Функциональность

Вторая вкладка нижней навигации — избранные магазины. Пользователь может:

- Добавить магазин в избранное нажатием на иконку сердца (на карточке магазина или на странице магазина)
- Просмотреть список избранных магазинов
- Быстро перейти к каталогу магазина из избранного
- Удалить магазин из избранного

## Хранение данных

Избранные магазины хранятся в Firestore в привязке к пользователю. При загрузке страницы избранного подгружается актуальная информация о каждом магазине (статус работы, время доставки).

Если пользователь не авторизован, избранное недоступно.
