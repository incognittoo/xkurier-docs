# Главная и магазины

## Главная страница

При открытии приложения пользователь попадает на главную страницу, которая содержит:

- **Баннеры** — карусель рекламных баннеров в верхней части экрана. Баннеры могут вести на веб-страницу или на страницу магазина.
- **Список магазинов** — карточки с логотипом, названием, адресом и временем доставки.

Данные главной страницы загружаются параллельно (баннеры и магазины) для быстрого отображения. Пока данные грузятся, показывается скелетон-анимация.

## Страница магазина

При нажатии на карточку магазина открывается его детальная страница:

- Информация о магазине (название, адрес, телефон, статус работы)
- График работы магазина по дням недели
- Время доставки (например, «30-45 мин»)
- Список товаров магазина с фильтрацией по категориям
- Возможность добавить магазин в избранное

## Статусы магазина

Магазин может находиться в одном из четырёх состояний:

| Статус | Описание |
|---|---|
| Открыт | Магазин работает, можно заказывать |
| Закрыт | Нерабочее время по расписанию |
| Временно закрыт | Вручную закрыт менеджером |
| Недоступен | Деактивирован администратором |

Если магазин закрыт, оформить заказ нельзя — пользователь увидит соответствующее сообщение.
