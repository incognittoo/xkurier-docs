# Клиентское приложение (xcourier)

Мобильное приложение для покупателей. Позволяет просматривать каталоги магазинов, выбирать товары, оформлять заказы и отслеживать их доставку.

## Для кого

Конечные пользователи — жители города, которые хотят заказать продукты с доставкой на дом или самовывозом.

## Основные возможности

- Просмотр списка магазинов и их каталогов
- Поиск товаров по названию
- Фильтрация товаров по категориям
- Корзина с подсчётом итоговой стоимости
- Оформление заказа с выбором адреса доставки и способа оплаты
- Отслеживание статуса заказа в реальном времени
- Избранные магазины
- Обращения в поддержку
- Push-уведомления о смене статуса заказа

## Навигация приложения

Приложение использует нижнюю панель с четырьмя вкладками:

| Вкладка | Описание |
|---|---|
| Главная | Список магазинов и баннеры |
| Избранное | Сохранённые магазины |
| Корзина | Товары для заказа |
| Профиль | Данные пользователя, история заказов, поддержка |

## Авторизация

Пользователь может пользоваться приложением анонимно (просмотр каталога), но для оформления заказа необходима аутентификация через Firebase Auth.
