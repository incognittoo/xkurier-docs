# Профиль

## Страница профиля

Четвёртая вкладка нижней навигации. Содержит:

- **Имя пользователя** и номер телефона
- **Мои заказы** — переход к истории заказов
- **Поддержка** — обращения в техподдержку
- **Пользовательское соглашение** — страница с текстом соглашения
- **Политика конфиденциальности** — страница с политикой обработки данных
- **Выход** — выход из аккаунта

## Данные пользователя

Профиль пользователя хранит:
- ID (Firebase Auth UID)
- Имя (displayName)
- Телефон
- Признак анонимности
- Адрес доставки (последний использованный)

Адрес доставки сохраняется автоматически при оформлении заказа и подставляется при следующем заказе.
