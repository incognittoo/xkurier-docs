# Поддержка

## Обращения в поддержку

Из профиля пользователь может перейти в раздел поддержки, где доступно:

### Список обращений
- Все обращения пользователя с текущим статусом
- Статусы: Открыт, В обработке, Решён, Закрыт

### Создание обращения
- Поле заголовка
- Поле описания проблемы
- После отправки обращение появляется в списке со статусом «Открыт»

### Детали обращения
- Полный текст обращения
- Текущий статус
- Ответ администратора (если есть)
- Дата создания и обновления

## Удаление обращения

Пользователь может удалить обращение, если оно не находится в статусе «В обработке».

## Уведомления

При ответе администратора на обращение пользователь получает push-уведомление через Cloud Function `onSupportTicketUpdated`.
